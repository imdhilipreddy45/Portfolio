# 3D Futuristic Portfolio - Dileep Reddy Devaram

A world-class futuristic 3D developer portfolio website built with React.js, Vite, Tailwind CSS, Framer Motion, and React Three Fiber.

## Features
- **3D Immersive Hero Section**: Interactive 3D objects using React Three Fiber.
- **Glassmorphism Design**: Modern frosted glass aesthetics with soft shadows.
- **Smooth Animations**: Powered by Framer Motion.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile viewing.
- **Custom Cursor**: Interactive cursor with hover effects.
- **Cinematic Loading Screen**: Gives a premium first impression.

## Tech Stack
- React.js + Vite
- Tailwind CSS
- Framer Motion
- React Three Fiber + Drei
- Lucide React & React Icons
- React Intersection Observer
- React Scroll

## Getting Started

1. **Install Dependencies**
```bash
npm install
```

2. **Run Development Server**
```bash
npm run dev
```

3. **Build for Production**
```bash
npm run build
```

## Deployment
This project is ready to be deployed on Vercel or Netlify. Just link your GitHub repository and build with `npm run build` using the `dist` directory.
